# NAME

whose - write the package COMMAND belongs to to standard ouput

# SYNOPSIS

whose *COMMAND*

# DESCRIPTION

whose writes the package COMMAND belongs to to standard ouput with the
help of which(1) and equery(1).

# SEE ALSO

`which`(1), `equery`(1).
